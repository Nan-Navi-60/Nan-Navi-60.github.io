- Database: MySQL, MariaDB, AWS RDS MySQL, Redis
- AI/ML: YOLOv9 (COCO + Custom), Google Gemini 2.0 Flash API
- 인증: 세션 기반 인증, JWT (쿠키 저장), Redis 세션 공유
- ORM/DB연동: MyBatis, JDBC 직접 연결, Django ORM, PyMySQL, HikariCP
- 인프라: Docker, Nginx (이중 부하분산), MySQL Cluster (GTID 복제), MySQL Router
- 디자인 도구: PhotoShop, Illustrator, AdobeXd
- 배포: AWS, GitHub Actions, GitHub Pages
